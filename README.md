# gitmoji by js

## About

[gitmoji](https://github.com/carloscuesta/gitmoji) 프로젝트를 한글화하고,
기존에 emoji가 찾아보기 힘든 점을 감안해 
실제 사용 시 필요한 분류로 분리하여 
보기 좋게 나누기 위해 작성된 프로젝트입니다.
